User Guide

Comprehensive Usage Instructions

Version 3.0 | January 2026

Overview

Ground Param is a 4-module program designed for analyzing geotechnical parameters from SPT (Standard Penetration Test) data:

  1. Module 1: SPT Plot - SPT data input and visualization
  2. Module 2: Laboratory Result Input - Laboratory test data integration
  3. Module 3: Parameters Summary - Calculation of geotechnical parameters
  4. Module 4: Multi-Parameters Plot - Multi-parameter comparison graphs

Module 1: SPT Plot

Purpose

Module 1 allows users to input SPT N-values and visualize them alongside soil classification, groundwater level, and effective stress profiles.

Key Features

1. Axis Settings Panel (Left Side)

  • Depth Range: Set minimum and maximum depth (meters)
  • N-value Range: Set SPT N-value display range (0-100)
  • Graph Title: Customize graph title
  • Label Size: Adjust text size (8-20 pt)

2. Data Entry Table

Input Methods:

  • Direct Entry: Click cell and type value
  • Paste from Excel: Copy data → Select top-left cell → Ctrl+V
  • Enter Key Navigation: Press Enter to move down to next row

Columns:

3. Functions

  • Add Row: Insert new data row at the end
  • Delete Row: Remove selected rows
  • Clear All: Delete all table data (confirmation required)
  • Send to Module 2: Transfer data to Module 2 for lab data entry
  • Update Graph: Refresh visualization with current data

4. Graph Display

The graph shows three panels:

  • Left Panel: Depth vs Soil Classification (USCS symbols)
  • Right Panel: Depth vs SPT N-value with groundwater level (blue dashed line)

Module 2: Laboratory Result Input

Purpose

Module 2 allows entry of laboratory test results to supplement SPT data, providing more accurate parameters for analysis.

Key Features

1. Data Synchronization

After clicking "Send to Module 2" in Module 1:

  • BH names and depths are automatically populated
  • USCS classifications are synchronized
  • Users only need to enter lab test results

2. Laboratory Parameters

3. Functions

  • Add Row: Insert additional test results
  • Delete Row: Remove selected data rows
  • Clear All: Delete all laboratory data
  • Send to Module 3: Transfer combined SPT and lab data for parameter calculation

Usage Notes

Important:

  • Lab data is optional - if not available, Module 3 will use SPT correlations
  • Lab results override SPT correlations when both are available
  • Leave cells empty if parameter was not tested

Module 3: Parameters Summary

Purpose

Module 3 calculates all geotechnical parameters using SPT correlations and/or laboratory test results.

Key Features

1. Calculation Settings

  • Structure Type: Select foundation type (affects K₀ calculation)
  • Surface Condition: Choose surface roughness (affects interface friction)
  • Alpha Value: Coefficient for Su calculation (0.06-0.12)
  • Beta Value: Coefficient for E calculation in sand (5-15)
  • k Value: Coefficient for E calculation in clay (200-500)

2. Parameters Calculated

3. Results Display

  • Results shown in table format for all boreholes and depths
  • Parameters marked with * indicate laboratory test values (not calculated)
  • Borehole-specific settings can be customized per BH

4. Functions

  • Calculate: Compute all parameters with current settings
  • Send to Module 4: Transfer results for visualization
  • Export Results: Save calculations to file

Module 4: Multi-Parameters Plot

Purpose

Module 4 creates comparison graphs showing multiple parameters across different boreholes.

Key Features

1. Loading Data

  • Click "Load Data from Module 3" to import calculated parameters
  • Select which boreholes to include in visualization
  • Data includes all parameters from Module 3

2. Parameter Selection

Select up to 8 parameters to display:

  • N-value, Ncor
  • USCS, γsat
  • σ'v, Su, φ'
  • E, ν, K₀, δ

3. Axis Customization

For each selected parameter, customize:

  • Min Value: Minimum axis value
  • Max Value: Maximum axis value
  • Grid: Show/hide grid lines
  • Y-axis Mode: Depth (downward) or Elevation (upward)

4. Reference Lines

Add vertical reference lines to any graph:

Add Row: Click "Add Row" button to insert new reference line

5. Functions

  • Update Graph: Refresh display with current settings
  • Export Graph: Save as PNG image
  • Clear Lines: Remove all reference lines

Save/Load Project System

Saving Projects

  1. Click File → Save Project (Ctrl+S)
  2. Choose location and filename (.json)
  3. All data from Modules 1-4 is saved

Saved Data Includes:

  • Module 1: SPT data, axis settings, graph configuration
  • Module 2: Laboratory test results
  • Module 3: Calculation settings, results, borehole-specific settings
  • Module 4: Selected parameters, axis limits, reference lines, plot configuration

Loading Projects

  1. Click File → Open Project (Ctrl+O)
  2. Select saved .json file
  3. All modules are populated with saved data
  4. Graphs are automatically updated

Workflow Example

Typical Analysis Workflow:

  1. Open Module 1 → Enter SPT data → Update Graph → Send to Module 2
  2. In Module 2 → Enter lab data (if available) → Send to Module 3
  3. In Module 3 → Set calculation parameters → Calculate → Review results → Send to Module 4
  4. In Module 4 → Load data → Select parameters → Customize axes → Add reference lines → Update Graph
  5. Save Project (Ctrl+S) for future reference

Important Notes

  • Always save your project regularly to prevent data loss
  • Module data flows sequentially: 1 → 2 → 3 → 4
  • Laboratory data in Module 2 overrides SPT correlations in Module 3
  • Reference lines in Module 4 display red value labels automatically
  • All calculations follow internationally recognized geotechnical correlations
  • Results should be verified by qualified geotechnical engineers

ภาพรวม

Ground Param เป็นโปรแกรม 4 โมดูล สำหรับวิเคราะห์พารามิเตอร์ทางปฐพีกลศาสตร์จากข้อมูล SPT:

  1. Module 1: SPT Plot - กรอกข้อมูล SPT และแสดงกราฟ
  2. Module 2: Laboratory Result Input - กรอกข้อมูลทดสอบในห้องปฏิบัติการ
  3. Module 3: Parameters Summary - คำนวณพารามิเตอร์ทางปฐพีกลศาสตร์
  4. Module 4: Multi-Parameters Plot - แสดงกราฟเปรียบเทียบหลายพารามิเตอร์

Module 1: SPT Plot

วัตถุประสงค์

Module 1 ใช้สำหรับกรอกข้อมูล SPT N-value และแสดงกราฟร่วมกับการจำแนกดิน ระดับน้ำใต้ดิน และหน่วยแรงประสิทธิผล

ฟังก์ชันหลัก

1. แผงตั้งค่ากราฟ (ด้านซ้าย)

  • ช่วงความลึก: กำหนดความลึกต่ำสุดและสูงสุด (เมตร)
  • ช่วง N-value: กำหนดช่วงแสดงผล SPT N-value (0-100)
  • ชื่อกราฟ: ปรับแต่งชื่อกราฟ
  • ขนาดตัวอักษร: ปรับขนาดข้อความ (8-20 pt)

2. ตารางกรอกข้อมูล

วิธีการกรอกข้อมูล:

  • พิมพ์โดยตรง: คลิกเซลล์และพิมพ์ข้อมูล
  • วางจาก Excel: คัดลอกข้อมูล → เลือกเซลล์มุมซ้ายบน → Ctrl+V
  • กด Enter: กด Enter เพื่อย้ายไปแถวถัดไป

คอลัมน์:

คอลัมน์คำอธิบายตัวอย่าง
BHชื่อหลุมเจาะBH-01
Depth (m)ความลึกเก็บตัวอย่าง1.5
N-valueค่า SPT15
USCSการจำแนกดินSM, CL, SW
GL. (m)ระดับปากหลุม100.0
WL. (m)ระดับน้ำใต้ดิน-2.0

3. ฟังก์ชัน

  • Add Row: เพิ่มแถวใหม่ท้ายตาราง
  • Delete Row: ลบแถวที่เลือก
  • Clear All: ลบข้อมูลทั้งหมด (ต้องยืนยัน)
  • Send to Module 2: ส่งข้อมูลไปยัง Module 2
  • Update Graph: อัพเดตกราฟตามข้อมูลปัจจุบัน

4. การแสดงกราฟ

กราฟประกอบด้วย 3 แผง:

  • แผงซ้าย: ความลึก vs การจำแนกดิน (สัญลักษณ์ USCS)
  • แผงขวา: ความลึก vs SPT N-value พร้อมระดับน้ำใต้ดิน (เส้นประสีน้ำเงิน)

Module 2: Laboratory Result Input

วัตถุประสงค์

Module 2 ใช้สำหรับกรอกผลการทดสอบในห้องปฏิบัติการเพื่อเสริมข้อมูล SPT ทำให้ได้พารามิเตอร์ที่แม่นยำยิ่งขึ้น

ฟังก์ชันหลัก

1. การซิงค์ข้อมูล

หลังจากกด "Send to Module 2" ใน Module 1:

  • ชื่อหลุมเจาะและความลึกจะถูกนำเข้าอัตโนมัติ
  • การจำแนกดิน USCS จะถูกซิงค์
  • ผู้ใช้เพียงกรอกผลการทดสอบจากห้องแล็บ

2. พารามิเตอร์จากห้องปฏิบัติการ

พารามิเตอร์คำอธิบายหน่วย
Suกำลังรับแรงเฉือนแบบไม่ระบายน้ำ (ดินเหนียว)kPa
φ'มุมเสียดทานประสิทธิผล (ทราย)องศา
EโมดูลัสของยังMPa
νอัตราส่วนปัวซอง-

3. ฟังก์ชัน

  • Add Row: เพิ่มแถวผลการทดสอบ
  • Delete Row: ลบแถวที่เลือก
  • Clear All: ลบข้อมูลห้องแล็บทั้งหมด
  • Send to Module 3: ส่งข้อมูล SPT + Lab ไปคำนวณพารามิเตอร์

หมายเหตุการใช้งาน

สำคัญ:

  • ข้อมูลห้องแล็บเป็นตัวเลือก - ถ้าไม่มี Module 3 จะใช้ SPT correlation
  • ผลห้องแล็บจะแทนที่ SPT correlation เมื่อมีข้อมูลทั้งสอง
  • ปล่อยช่องว่างถ้าไม่ได้ทดสอบพารามิเตอร์นั้น

Module 3: Parameters Summary

วัตถุประสงค์

Module 3 คำนวณพารามิเตอร์ทางปฐพีกลศาสตร์ทั้งหมดจาก SPT correlation และ/หรือผลการทดสอบห้องปฏิบัติการ

ฟังก์ชันหลัก

1. ตั้งค่าการคำนวณ

  • Structure Type: เลือกประเภทฐานราก (ส่งผลต่อค่า K₀)
  • Surface Condition: เลือกความขรุขระของผิว (ส่งผลต่อมุมเสียดทานระหว่างผิว)
  • Alpha Value: ค่าสัมประสิทธิ์สำหรับคำนวณ Su (0.06-0.12)
  • Beta Value: ค่าสัมประสิทธิ์สำหรับคำนวณ E ในทราย (5-15)
  • k Value: ค่าสัมประสิทธิ์สำหรับคำนวณ E ในดินเหนียว (200-500)

2. พารามิเตอร์ที่คำนวณได้

พารามิเตอร์คำอธิบายวิธีการคำนวณ
σ'vหน่วยแรงประสิทธิผลในแนวดิ่งσv - u
CNค่าปรับแก้หน่วยแรงบรรทุก√(100/σ'v)
Ncorค่า N ที่ปรับแก้แล้วCN × Nfield
Suกำลังรับแรงเฉือนแบบไม่ระบายน้ำα × Ncor × Pa (หรือ Lab)
φ'มุมเสียดทานประสิทธิผล27.1+0.3Ncor-0.00054Ncor² (หรือ Lab)
Eโมดูลัสของยังk×Su หรือ β×Ncor×Pa (หรือ Lab)
νอัตราส่วนปัวซองตามประเภทดิน (หรือ Lab)
K₀ค่าสัมประสิทธิ์แรงดันดินขณะหยุดนิ่ง1-sin(φ') หรือ 0.95-sin(φ')
Rintค่าลดทอนระหว่างผิวตามประเภทพื้นผิว
δมุมเสียดทานระหว่างผิวRint × φ'

3. การแสดงผล

  • ผลลัพธ์แสดงในรูปแบบตารางสำหรับทุกหลุมเจาะและความลึก
  • พารามิเตอร์ที่มีเครื่องหมาย * คือค่าจากห้องปฏิบัติการ (ไม่ได้คำนวณ)
  • สามารถปรับแต่งการตั้งค่าเฉพาะหลุมเจาะได้

4. ฟังก์ชัน

  • Calculate: คำนวณพารามิเตอร์ทั้งหมด
  • Send to Module 4: ส่งผลลัพธ์ไปแสดงกราฟ
  • Export Results: บันทึกผลการคำนวณ

Module 4: Multi-Parameters Plot

วัตถุประสงค์

Module 4 สร้างกราฟเปรียบเทียบหลายพารามิเตอร์ของหลุมเจาะต่างๆ

ฟังก์ชันหลัก

1. โหลดข้อมูล

  • คลิก "Load Data from Module 3" เพื่อนำเข้าพารามิเตอร์ที่คำนวณแล้ว
  • เลือกหลุมเจาะที่ต้องการแสดง
  • ข้อมูลรวมพารามิเตอร์ทั้งหมดจาก Module 3

2. เลือกพารามิเตอร์

เลือกพารามิเตอร์ได้สูงสุด 8 ตัว:

  • N-value, Ncor
  • USCS, γsat
  • σ'v, Su, φ'
  • E, ν, K₀, δ

3. ปรับแต่งแกน

สำหรับแต่ละพารามิเตอร์ที่เลือก สามารถปรับแต่ง:

  • Min Value: ค่าต่ำสุดของแกน
  • Max Value: ค่าสูงสุดของแกน
  • Grid: แสดง/ซ่อนเส้นตาราง
  • Y-axis Mode: ความลึก (ลงล่าง) หรือ ระดับ (ขึ้นบน)

4. เส้นอ้างอิง

เพิ่มเส้นแนวตั้งอ้างอิงในกราฟใดก็ได้:

คอลัมน์คำอธิบาย
Graphเลือกกราฟพารามิเตอร์ (1-8)
Valueค่าบนแกน X สำหรับเส้นแนวตั้ง
From (m)ความลึกเริ่มต้น
To (m)ความลึกสิ้นสุด
Colorสีเส้น (รหัสสี hex)
Line StyleSolid, Dashed, Dotted, Dash-dot
Widthความหนาเส้น (1-5)
Labelป้ายข้อความ (ไม่บังคับ)

Add Row: คลิกปุ่ม "Add Row" เพื่อเพิ่มเส้นอ้างอิงใหม่

5. ฟังก์ชัน

  • Update Graph: อัพเดตกราฟตามการตั้งค่าปัจจุบัน
  • Export Graph: บันทึกเป็นรูป PNG
  • Clear Lines: ลบเส้นอ้างอิงทั้งหมด

ระบบบันทึก/เปิดโปรเจค

การบันทึกโปรเจค

  1. คลิก File → Save Project (Ctrl+S)
  2. เลือกตำแหน่งและชื่อไฟล์ (.json)
  3. ข้อมูลทั้งหมดจาก Module 1-4 จะถูกบันทึก

ข้อมูลที่บันทึก:

  • Module 1: ข้อมูล SPT, การตั้งค่ากราฟ, การตั้งค่าแกน
  • Module 2: ผลการทดสอบจากห้องปฏิบัติการ
  • Module 3: การตั้งค่าการคำนวณ, ผลลัพธ์, การตั้งค่าเฉพาะหลุมเจาะ
  • Module 4: พารามิเตอร์ที่เลือก, ขอบเขตแกน, เส้นอ้างอิง, การตั้งค่ากราฟ

การเปิดโปรเจค

  1. คลิก File → Open Project (Ctrl+O)
  2. เลือกไฟล์ .json ที่บันทึกไว้
  3. โมดูลทั้งหมดจะถูกเติมข้อมูลที่บันทึกไว้
  4. กราฟจะอัพเดตอัตโนมัติ

ตัวอย่างขั้นตอนการทำงาน

ขั้นตอนการวิเคราะห์ทั่วไป:

  1. เปิด Module 1 → กรอกข้อมูล SPT → อัพเดตกราฟ → ส่งไป Module 2
  2. ใน Module 2 → กรอกข้อมูลห้องแล็บ (ถ้ามี) → ส่งไป Module 3
  3. ใน Module 3 → ตั้งค่าการคำนวณ → คำนวณ → ตรวจสอบผล → ส่งไป Module 4
  4. ใน Module 4 → โหลดข้อมูล → เลือกพารามิเตอร์ → ปรับแต่งแกน → เพิ่มเส้นอ้างอิง → อัพเดตกราฟ
  5. บันทึกโปรเจค (Ctrl+S) เพื่ออ้างอิงในอนาคต

ข้อควรทราบ

  • บันทึกโปรเจคเป็นประจำเพื่อป้องกันข้อมูลสูญหาย
  • ข้อมูลไหลแบบต่อเนื่อง: Module 1 → 2 → 3 → 4
  • ข้อมูลห้องแล็บใน Module 2 จะแทนที่ SPT correlation ใน Module 3
  • เส้นอ้างอิงใน Module 4 จะแสดงค่าเป็นป้ายสีแดงอัตโนมัติ
  • การคำนวณทั้งหมดอิงตาม correlation ทางปฐพีกลศาสตร์ที่ได้รับการยอมรับในระดับสากล
  • ผลลัพธ์ควรได้รับการตรวจสอบโดยวิศวกรปฐพีกลศาสตร์ที่มีคุณสมบัติ
